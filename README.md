# ホテル清掃シフト最適化ダッシュボード

## プロジェクト概要
ホテル業界7年の実務経験をもとに、チェックアウト時間・客層・シーズンなどの
予約データを分析し、清掃スタッフの最適なシフト配置を提案するダッシュボード。

## 使用技術
- Python
- pandas
- matplotlib
- SQLite / SQL
- Streamlit

## 分析の背景
小規模ホテルでの実務経験から、清掃シフトの人員配置は
現場感覚に頼ることが多く、データによる裏付けが少ない課題があった。
本プロジェクトでは、予約データを可視化することで
効率的なシフト設計の根拠を提供することを目的とする。

## 機能説明
- 月別・シーズン別・曜日別の予約件数の可視化
- チェックアウト時間帯・客層別の傾向分析
- 清掃スタッフの月別必要人数の算出

## 前提条件
- 清掃開始時間：8時以降
- スタッフ1人の稼働時間：240分（休憩なし）
- 清掃所要時間は実務経験をもとに設定
